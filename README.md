Hi, I'm Natthapong Thaboran. 
I'm a Software Developer (kind of) who can design a bit. Currently working at BAAC while also taking freelance UX/UI Designer projects.
I started in UX/UI, then moved to frontend with React and Next.js. Now I work on both mobile and backend — using Flutter, C#, and .NET Core to build full microservice mobile apps, including deployment to the stores.
I also handle websites, APIs, and services using Docker, Vercel, Railway, and internal servers. For databases, I work with Supabase, PostgreSQL, and SQL Server.
Outside of coding, I enjoy design, listening to music, playing games, watching series, and sleeping all day 😌
